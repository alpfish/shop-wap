# Waves - 指令
> MDL风格，元素波纹效果。 [代码来源](https://github.com/Teddy-Zhu/vue-waves)；
> 和 ui/components/ripple 组件效果一样，但用法不同，更简单


## 用法
```javascript
import Vue from 'vue'
import VueWaves from 'ui/directive/waves'
Vue.use(VueWaves)
```
```html
<!-- 在元素上使用，建议在简单层次的元素中使用 -->
<teamplate>
   <div v-waves>
</teamlate>
```

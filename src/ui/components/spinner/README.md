# Spinner

载入动画

----

## 用法

```html
<spinner type="android"></spinner>
```

## Props
| 参数 | 说明 |	类型 | 可选值 | 默认值 |
| :---- | :---- | :---- | :---- | :---- |
| type | 类型 | String | 'android', 'ios', 'ios-small', 'bubbles', 'circles', 'crescent', 'dots', 'lines', 'ripple', 'spiral' | ios |
| color | 彩色 | String | | |
| size | 大小 | String | | |

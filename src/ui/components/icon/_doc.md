# Icon

图标

> 字体图标及样式名称来源于 iconfont.cn

----

## 用法

```html
<icon name="gouwuche" :size="18"></icon>
```

## Props
| 参数 | 说明 |	类型 | 可选值 | 默认值 |
| :---- | :---- | :---- | :---- | :---- |
| name | 图标名称（图标下载后见demo页） | String |  | 无 |
| size | 图标大小 | Number |  | 24 |
| color | 图标颜色 | String |  |  |

## 新增图标
到 iconfont.cn 下载所需图标（必须保留原来已使用的图标），解压后覆盖 src/assets/iconfont/ 目录即可。

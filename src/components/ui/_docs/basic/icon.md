# Icon


图标组件，使用 iconfont 中的的图标字体

------

## 用法

```html
<icon name="arrow_back"></icon>
<icon name="info" size="24"></icon>
```

## API

| 参数 | 说明 |	类型 | 可选值 | 默认值 |
| :---- | :---- | :---- | :---- | :---- |
| name  | icon 名称  | String |  |  |
| size   | icon 大小 | Number |  | 24 |
